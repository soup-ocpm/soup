Graph Creation Methods

You have two options for creating the graph.

Standard method

This will run standard queries within the Memgraph database. This operation will take longer if the .csv file is large.

Load CSV method

This operation will be more suitable if the csv file is large, since it uses the query "LOAD CSV..."

Warning :

This operation will copy (temporarily) the .csv file inside the Docker Container which will be specified in the next screen.
The file will be saved in a "/temp" folder and will be correctly deleted once the LOAD operation is finished.
